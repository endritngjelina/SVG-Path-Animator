# 🎨 SVG Path Animator

A progressive collection of standalone SVG path animation experiments built with pure HTML, CSS, and JavaScript — no frameworks or dependencies required.

Each version (0–14) builds upon the last, exploring different animation techniques, visual effects, and interaction patterns using the SVG `<path>` element.

---

## 🚀 Features

- Pure vanilla HTML, CSS, and JavaScript — zero dependencies
- 15 progressively enhanced animation versions
- SVG stroke-dasharray / stroke-dashoffset animation techniques
- Smooth path drawing and self-drawing effects
- Lightweight, browser-ready `.html` files — just open and run

🧠 How It Works
SVG paths are animated using the stroke-dasharray and stroke-dashoffset CSS properties. By dynamically adjusting the offset, paths appear to "draw themselves" over time. JavaScript calculates total path length using path.getTotalLength() for precise, dynamic control.
